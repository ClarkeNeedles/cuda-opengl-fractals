# Standalone CUDA-OpenGL Mandelbrot Benchmark Suite

## High-Level Overview
This project is a standalone, high-performance Mandelbrot fractal renderer leveraging NVIDIA CUDA and OpenGL. Designed as a companion to an existing WinAPI CPU/SIMD fractal generator, its primary purpose is to benchmark and compare rendering performance between specialized CPU vector units and raw GPU cores. The application emphasizes direct GPU rendering and dynamic interaction for exploring the Mandelbrot set.

## System Architecture
*   **Standalone Design**: The project maintains complete separation from any existing WinAPI projects to ensure configuration independence and avoid dependency conflicts.
*   **Continuous Rendering Loop**: A decoupled main `while` loop runs continuously, independent of input callbacks. It targets 60+ FPS, ensuring the complex plane is completely redrawn on the GPU every frame during user interaction.
*   **CUDA-OpenGL Interoperability**: Maximizes performance using Pixel Buffer Objects (PBOs) for zero-copy data transfer. Pixel color data is generated by CUDA kernels, then registered with OpenGL via `cudaGraphicsGLRegisterBuffer` and written directly to the GPU's texture memory. The interop boilerplate ensures the pointer is mapped before kernel launch and unmapped immediately after execution, bypassing CPU involvement in pixel data transfer.
*   **Dynamic Precision Kernels**: Features a unified template CUDA kernel supporting both `float` and `double` precision math blocks. The application dynamically tracks the zoom factor and hot-swaps to the `double` precision kernel once rendering precision limits (e.g., `1e-7`) are reached, preserving fractal detail at deep zoom levels.
*   **Interactive Control**: Mouse event deltas (drag and scroll) are mapped directly to complex coordinate shifts. Scale adjustments are applied to ensure the fractal remains locked perfectly to the cursor position during pans and zooms.
*   **Parallel Mapping**: Individual 2D thread IDs (`(x, y)` grid) are directly mapped to physical pixel addresses and their corresponding coordinates on the complex plane for efficient parallel computation.

## Tech Stack and Dependencies
*   **GPU Computing**: NVIDIA CUDA
*   **Graphics API**: OpenGL
*   **Windowing Library**: GLFW (or SDL2)
*   **OpenGL Extension Management**: GLEW/GLAD
*   **Build System**: CMake or Visual Studio (with CUDA build customizations enabled)

## Features
*   **High-Performance Rendering**:
    *   Achieves a continuous frame loop targeting 60+ FPS.
    *   Leverages CUDA-OpenGL Interoperability via Pixel Buffer Objects (PBOs) for direct GPU texture writing, ensuring zero-copy data transfer and maximized rendering performance.
    *   Efficient parallel computation by mapping 2D thread IDs directly to pixel addresses and complex plane coordinates.
*   **Dynamic Precision**:
    *   Supports both `float` and `double` precision for Mandelbrot iterative calculations.
    *   Automatically hot-swaps between the fast `float` path and the deeper `double` path based on zoom depth (e.g., beyond `1e-7` precision) to maintain fractal detail at high magnification.
*   **Interactive Exploration**:
    *   Smooth zoom and drag-to-pan functionality for intuitive exploration of the Mandelbrot set.
    *   Mouse events (drag, scroll) directly modify `centerX`, `centerY`, and `zoom` uniform parameters.
    *   Fractal remains locked perfectly to the cursor's position during interactive pans and zooms for a consistent user experience.

## Prerequisites / System Requirements
To build and run this project, the following are required:
*   **GPU**: NVIDIA GPU with CUDA compute capability.
*   **Drivers**: Up-to-date NVIDIA display drivers.
*   **SDK**: NVIDIA CUDA Toolkit installed.
*   **Operating System**: A Windows environment is implied for Visual Studio configurations, though CMake can be used cross-platform.
*   **Build Tools**: CMake or Visual Studio (with CUDA build customizations).

## Quick Start / Usage Examples
Once the application is compiled and launched, it continuously renders the Mandelbrot set in a dedicated window.
*   **Pan**: Click and drag the mouse anywhere on the fractal to pan across the complex plane. The fractal will smoothly follow your cursor's movement.
*   **Zoom**: Use the mouse scroll wheel to zoom in or out. The fractal will zoom towards or away from the cursor's current position, maintaining focus on the area of interest.
The rendering frame rate is designed to be above 60 FPS, providing a fluid and highly interactive experience for exploring the fractal.
